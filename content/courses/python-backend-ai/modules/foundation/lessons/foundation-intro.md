# Введение

Этот урок задаёт рабочий контекст перед практикой.

- Держим фокус на маленьких проверяемых шагах.
- После каждого шага фиксируем понятный результат.

Этого достаточно, чтобы пройти модуль без потери темпа и без лишнего scope.
