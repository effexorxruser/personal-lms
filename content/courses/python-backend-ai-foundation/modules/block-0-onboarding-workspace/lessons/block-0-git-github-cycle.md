# Block 0 · Git/GitHub cycle

## Why this matters (RU)
Без Git/GitHub твой прогресс нельзя проверять и демонстрировать. Этот урок превращает локальные шаги в внешний проверяемый результат.

## What to read (EN source)
- Git Docs: `git init`, `git status`, `git add`, `git commit`.
- GitHub Skills: упражнение на базовый workflow с commit и публикацией изменений.

## What to skip
Пропусти темы rebase, squash, cherry-pick, advanced merge conflict resolution и настройку CI. Для Block 0 нужен только базовый publish cycle.

## Action
Инициализируй репозиторий (или используй существующий), добавь файлы `scripts/hello_cli.py` и `notes/foundation-log.txt`, сделай commit и push в GitHub. Сохрани ссылку на commit/PR в submission.

## Definition of Done
- В репозитории есть commit, связанный с артефактами Block 0.
- Ссылка на commit или PR открывается для review.
- В описании submission коротко объяснено, что именно зафиксировано.
