# Block 0 · Weekly learning loop setup

## Why this matters (RU)
Курс на 6 месяцев выигрывается не мотивацией, а системностью. Weekly learning loop снижает хаос и делает прогресс измеримым каждую неделю.

## What to read (EN source)
- Python MOOC: введение о структуре курса и принципе регулярной практики.
- Automate the Boring Stuff: Preface/intro о практическом подходе «learn by doing».

## What to skip
Не уходи в длинные теоретические главы. Пропусти всё, что не помогает прямо сейчас спланировать ритм учебных сессий и практики.

## Action
Сделай файл `notes/weekly-learning-loop.md`: укажи 4-6 сессий на неделю (день, длительность, фокус), опиши стартовый ритуал (2-3 шага) и закрывающий ритуал (2-3 шага), добавь fallback-правило на случай пропуска.

## Definition of Done
- В weekly-плане есть конкретные дни и длительность каждой сессии.
- Есть явный шаблон начала и завершения сессии.
- Есть правило восстановления ритма после пропуска.
