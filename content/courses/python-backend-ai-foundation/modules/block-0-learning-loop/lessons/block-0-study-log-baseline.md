# Block 0 · Study log baseline

## Why this matters (RU)
Учебный журнал — это твой operational memory. Он помогает не терять контекст между сессиями и ускоряет review со стороны ментора/агента.

## What to read (EN source)
- Python MOOC: разделы с практическими упражнениями, где важна фиксация результата.
- Automate the Boring Stuff: вводные части про практику через маленькие законченные шаги.

## What to skip
Пропусти расширенные проекты и длинные объяснения деталей языка. Цель урока — дисциплина фиксации результата, а не углубление в новую теорию.

## Action
Создай `notes/study-log.md` с шаблоном записи: дата, цель сессии, выполненные команды/шаги, наблюдаемый результат, blocker, следующий шаг. Заполни минимум 3 записи на основе работы в Block 0.

## Definition of Done
- Есть минимум 3 полноформатные записи по шаблону.
- В каждой записи есть наблюдаемый результат, а не абстрактная формулировка.
- Есть минимум один зафиксированный blocker с конкретным способом решения.
