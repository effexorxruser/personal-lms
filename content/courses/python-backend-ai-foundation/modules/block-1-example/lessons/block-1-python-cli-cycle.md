# Block 1 · Python CLI cycle

## Why this matters (RU)
Execution-first подход работает только если ты регулярно превращаешь чтение в конкретные маленькие действия.

## What to read (EN source)
- Python docs: https://docs.python.org/3/tutorial/introduction.html

## What to skip
Не уходи в большие туториалы и фреймворки на этом шаге.

## Action
Выполни в REPL `2 + 2`, затем создай `scripts/hello_cli.py` с `print("foundation")` и запусти файл.

## Definition of Done
- В run-log зафиксирован вывод REPL.
- Скрипт запускается без ошибок.
- В заметке есть 2–3 предложения про смысл шага.
