# Зачем этот шаг в маршруте

Checkpoint соединяет отдельные навыки в один маленький результат. Это не новый большой проект, а компактный integration-step foundation-блока.

## Что сделать

Собери `foundation-start-pack`:

- `scripts/hello_cli.py` с рабочим `--name`;
- `README.md` с разделами `Как запустить`, `Пример запуска`, `Что дальше`;
- короткий `run log` (можно в `notes/foundation-log.txt`) с 2-3 командами и наблюдаемым результатом.

## Какой объём достаточен

- Один скрипт.
- Один понятный сценарий запуска.
- Один репозиторий/ветка со ссылкой для review.

## Что считаем done

- Любой человек может повторить запуск по README за 2-3 минуты.
- Результат демонстрирует execution, а не только чтение материалов.
- Артефакт готов к отправке в checkpoint формы курса.
